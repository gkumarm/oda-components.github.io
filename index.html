ODA Component SDKs

Open Digital Architecture (ODA) Component software development kits (SDK), for Rust, Golang, and Node.js, provide tools, libraries, and resources for developers to build applications or services based on the principles of ODA. These SDKs enable developers to interact with and leverage the capabilities of the ODA ecosystem, promoting interoperability and seamless integration.

ODA Component RIs

Open Digital Architecture (ODA) Component reference implementations (RIs) are built on our ODA Component SDKs and provide concrete and functional examples demonstrating best practices. Our open source RIs serve as a starting point for developers looking to build ODA compliant solutions.

ODA Canvas SDKs

An Open Digital Architecture (ODA) Canvas software development kit (SDK) is a set of development tools designed to empower developers to create platforms and services in support of ODA Components.

ODA Canvas RIs

An Open Digital Architecture (ODA) Canvas reference implementation (RI) provides a platform, built on an ODA Canvas SDK, for hosting ODA Components. An RI provides a functional environment, and supporting services, demonstrating best practices. Our open source RIs serve as a starting point for developers looking to build ODA Canvas solutions.
